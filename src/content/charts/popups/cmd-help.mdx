Lists every command available in your session right now, including ones added by plugins and MCP servers, which no static reference can predict.

Typing `/` on its own does something similar and more useful day to day: it opens a filterable menu of commands and skills as you type.

Reach for `/help` when you suspect a command exists but cannot recall its name. There is no lesson for it because there is nothing to learn: it is the escape hatch that shows you the current truth of your own install.
